# Aquaculture Pond Detection Using Satellite Imagery

## Project Overview

This project aims to build a machine learning model capable of identifying aquaculture ponds from satellite-derived features.

The solution is developed for a machine learning competition where the model must generalize to unseen time periods while achieving strong F1-score and ROC-AUC performance.

The project follows professional machine learning and software engineering practices, including modular code, reproducible experiments, version control, and clear documentation.
